Introducción a las Redes I

ofrece una introducción fundamental a los conceptos, tecnologías y protocolos que permiten la comunicación entre computadoras en una red. A lo largo del curso, los estudiantes aprenderán sobre el modelo OSI, el modelo TCP/IP, direccionamiento IP, tipos de redes (LAN, WAN, MAN), y los principales dispositivos de red, como routers y switches. Además, se abordarán principios de seguridad en redes y técnicas básicas de monitoreo y diagnóstico. Este curso es esencial para comprender el funcionamiento de Internet y las comunicaciones en sistemas de redes modernas.


Temario del Curso

1. Introducción a las Redes de Computadoras

  • Definición de redes
  • Historia y evolución de las redes
  • Clasificación de redes
  • Ventajas de las redes

2. Modelo OSI y Modelo TCP/IP

  • Modelo OSI: Capas y Funciones
  • Modelo TCP/IP: Capas y Comparación con OSI
  • Protocolo IP y su función en la red
  • Protocolos de cada capa

3. Direccionamiento IP

  • Direcciones IPv4 e IPv6
  • Clases de direcciones IP
  • Subredes y máscaras de red
  • Asignación de direcciones: estática y dinámica

4. Equipos de Redes

  • Routers
  • Switches
  • Access Points
  • Firewalls

5. Protocolos de Comunicación

  • Protocolo TCP y UDP
  • Protocolo ARP
  • Protocolo ICMP
  • Protocolo DHCP
  • Protocolo DNS

6. Tipos de Redes

  • LAN (Red de Área Local)
  • WAN (Red de Área Amplia)
  • MAN (Red de Área Metropolitana)
  • WLAN (Red de Área Local Inalámbrica)

7. Seguridad en Redes

  • Principios básicos de seguridad
  • Criptografía y cifrado
  • Autenticación y autorización
  • Firewall y control de acceso

8. Monitoreo y Diagnóstico de Redes

  • Herramientas de monitoreo
  • Detección de fallos
  • Solución de problemas comunes en redes
